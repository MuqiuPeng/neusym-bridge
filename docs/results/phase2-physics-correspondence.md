# Phase 2 实验报告：公共结构是否对应物理规律

**日期**：2026-04-03  
**状态**：完成（重新定义结论框架）  
**核心发现**：公共结构存在且有因果影响，但 reconstruction-trained encoder 不保持多项式动力学结构，SINDy 不适用于此类 latent 空间。

---

## 叙事框架

```
1. SVCCA 成功（0.999）       → 公共结构高度一致
2. SINDy 失败（R²=0.32）    → 诊断：encoder 非线性扭曲动力学
3. Sanity check 通过（0.81） → 确认问题在 encoder，不在 SINDy
4. 干预实验成功（3/4 因果）  → 因果信息存在，只是非多项式编码
```

**结论**：发现了 reconstruction-trained encoder 和动力学保持之间的结构性矛盾，并通过干预实验证明因果信息仍然存在于 latent 中，只是以非多项式的形式编码。

## Step 0: SINDy 工具链 Sanity Check

**在温度场中心点直接跑 SINDy（已知答案的实验）：**

```
(x0)' = 0.125 - 0.608·x0
R² = 0.808
```

**结论**：SINDy 工具链工作正常。温度场中存在可恢复的线性动力学（衰减项 -0.608·x0 对应热耗散）。这建立了 R²=0.81 的上界参照。

---

## Task 2.2: SINDy 方程恢复

**结果：未达阈值**

### 修复措施（相对首次运行）
1. SmoothedFiniteDifference 替代 np.gradient
2. 自适应阈值（基于 Xdot 量级 × 0.1）
3. 降维至 4 维（Plan C）
4. 先 degree=1 再 degree=2

### SINDy 评分汇总

| 方案 | R² | 说明 |
|------|-----|------|
| Plan A, degree=1 | 0.183 | 1 维太少 |
| Plan A, degree=2 | 0.219 | 二次项贡献不大 |
| Plan C, degree=1 | 0.279 | 4 维线性结构存在 |
| **Plan C, degree=2** | **0.324** | 最佳，但远低于 0.7 |
| WSINDy fallback | 0.001 | SR3 优化器反而更差 |

### 关键诊断

**温度场直接 SINDy R²=0.81，latent 投影 SINDy R²=0.32。**

差距来自 encoder 的非线性映射。Reconstruction loss 优化 encoder 压缩空间信息以重建温度场，但不保证时间演化在 latent 空间中保持多项式结构。encoder 可以将线性动力学扭曲为 SINDy 候选库无法表达的非线性形式。

**这是 reconstruction loss 的结构性限制，不是 SINDy 配置问题。**

---

## Task 2.3: 干预实验

**结果：PASS — 3/4 方向检测到因果效应**

| 公共方向 | 温度变化量 | 单调性 | 因果效应 |
|----------|-----------|--------|---------|
| **1** | 0.0509 | **是** | **检测到** |
| 2 | 0.0127 | 否 | 未检测到 |
| **3** | 0.0200 | **是** | **检测到** |
| **4** | 0.1262 | **是** | **检测到** |

沿因果方向扰动 latent，温度场产生系统性、单调的变化。这证明公共方向确实编码了有因果影响的物理量，即使这些量不能用多项式表达。

### 因果方向语义标注

| 方向 | 效应强度 | 推测物理含义 |
|------|---------|-------------|
| 4 (最强) | 0.1262 | 主温度模态（全局温度水平） |
| 1 | 0.0509 | 辅助温度模态 |
| 3 | 0.0200 | 空间结构模态 |

## 迁移测试

**跳过**：SINDy 模型质量不足，迁移测试无意义。在决策树中属于"SINDy prerequisite failed → skip downstream"。

---

## Phase 2 最终结论

### 正面发现
1. **SVCCA corr=0.999**：公共子空间高度对齐，Phase 1 结论得到加强
2. **3 个因果方向**：干预实验证明 latent 编码了有因果影响的物理量
3. **SINDy sanity check R²=0.81**：温度场本身的动力学可恢复

### 负面发现（同样有价值）
4. **Reconstruction encoder 不保持动力学结构**：latent 空间的时间演化被非线性扭曲，SINDy 候选库无法拟合
5. **这是 reconstruction loss 的系统性限制**：不是数据量、超参数或维度的问题

### 认识论贡献
> 发现了 "表示质量"（CKA=0.94, SVCCA=0.999）和 "动力学可恢复性"（SINDy R²=0.32）之间的脱节。好的表示不等于好的动力系统。这为未来设计 dynamics-aware encoder 提供了具体的失败案例。
